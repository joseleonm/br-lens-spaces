About the database

The file data/BR_polynomials.json contains the full BR polynomial of the Heegaard graph G(p; ±1, ±q) on T² as a list of monomials \(x^s y^t z^{2g}\), together with the invariants S2, m2, F, G and the S-profile T0, for all 45 canonical orbits of L(p, q) under orientation-preserving homeomorphism with 3 ≤ p ≤ 20. The signature (S2, m2, F, G) distinguishes all orbits and resolves the 24 pairs with equal (S2, m2) found for p ≤ 100.

Computational complexity

The computation enumerates all 22p spanning subgraphs, reduced by Z/pZ symmetry to necklace classes. Cases p = 18, 19, 20 (★ HPC) required the CIMAT–Mérida supercomputer.

Reproducing the computation

# Build the Cython extension (recommended for p ≥ 13)
cd src/ && sage -python setup_BR.py build_ext --inplace

# Recompute all polynomials
sage src/BR_lens_parallel.sage

# Regenerate the HTML explorers from the JSON
python src/generate_explorers.py
