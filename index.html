Bollobás–Riordan Polynomial Database
Lens Spaces L(p, q) via Heegaard Ribbon Graphs

Interactive explorers for the Bollobás–Riordan polynomial of the Heegaard graph G(p; ±1, ±q) on the torus T². Each explorer lets you select spanning subgraphs and inspect the corresponding monomial, the ribbon-graph invariants, and the torus embedding in real time.

How to use an explorer

Left-click on an edge to add it to the subgraph F. Right-click to remove it. Use Complete G to select all 2p edges. The panel on the right shows:

  • the monomial Xs yt z2g(F) contributed by F,
  • ribbon-graph invariants k(F), f(F), g(F),
  • the coefficient of that monomial in BR(G),
  • the z-slice heat-map of the full polynomial at the selected genus level.

Repository contents

PathDescription
data/BR_polynomials.json Primary database: full BR polynomial (as term list) for each case, plus S-profile and invariants S₂, m₂, F, G
src/generate_explorers.py Python script that regenerates all HTML explorers from the JSON
src/BR_lens_parallel.sage SageMath script: parallel computation of BR polynomials via multiprocessing + Cython
src/BR_lens_worker.py Worker module used by the parallel script
src/BR_lens_core.pyx Cython extension: hot loop for ribbon-graph face counting (~1M masks/s per core)
src/setup_BR.py Build script for the Cython extension

# Install dependencies
pip install numpy plotly

python src/generate_explorers.py
Data computed with SageMath 10.7 + Cython on an 8-core Mac mini (Apple M2). Ribbon-graph model: Heegaard splitting of the lens space L(p,q); graph G(p; ±1, ±q) has p vertices and 2p edges on T². — Jose L. Leon — github.com/joseluis/br-lens-spaces
