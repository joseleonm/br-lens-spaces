Bollobás–Riordan Polynomial of Lens Spaces

For each lens space L(p, q), the canonical Heegaard ribbon graph G(p; ±1, ±q) is embedded on the torus T². This database contains the full BR polynomial of G, computed for all 45 homeomorphism orbits with 3 ≤ p ≤ 20. Each case comes with an interactive graph explorer for visualising spanning subgraphs and their polynomial contributions in real time.

About the database

The file data/BR_polynomials.json contains, for each of the 45 canonical orbits of L(p, q) under orientation-preserving homeomorphism (3 ≤ p ≤ 20), the following data: the full list of monomials \(x^s y^t z^{2g}\) of the Bollobás–Riordan polynomial of G(p; ±1, ±q), together with the crest invariants S2 and m2 extracted from the z2-slice of BR(G).

Computational complexity

The computation enumerates all 22p spanning subgraphs, reduced by Z/pZ symmetry to necklace classes. Cases p = 18, 19, 20 (★ HPC) required the CIMAT–Mérida supercomputer.

Reproducing the computation

# Build the Cython extension (recommended for p ≥ 13)
cd src/ && sage -python setup_BR.py build_ext --inplace

# Recompute all polynomials
sage src/BR_lens_parallel.sage

# Regenerate the HTML explorers from the JSON
python generate_explorers.py
